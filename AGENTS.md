# Çalışma notları

- Lobi kapasitesi 2–5 oyuncudur; hem istemci hem Convex doğrulaması korunmalıdır.
- Kritik puan, sıra, süre ve cevap kuralları yalnızca Convex tarafında uygulanır.
- Türkçe metin ve locale dönüşümlerinde `tr` locale kullanılmalıdır.
- Teslimden önce `npm test`, `npm run typecheck` ve `npm run build` çalıştırılmalıdır.
